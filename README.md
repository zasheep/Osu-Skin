# Osu-Skin
Skins for Osu 

Oregairu.osk : An Osu! skin made based off Yahari Ore no Seishun Love Comedy wa Machigatteiru.

Changes include:
  Combo Burst, 
  Colour, 
  Fail Screen, 
  Hit Circles, 
  Pause Screen, 
  Mode, 
  Mods, 
  Section Pass, 
  Section Fail, 
  Score Bar, 
  Song Selection Button, 
  Spinner, 

Not Included:
  Changes to other Osu! modes.
  Ranking-S,A,B..
